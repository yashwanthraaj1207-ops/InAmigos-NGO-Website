# 🌟 InAmigos NGO Website

A responsive NGO website developed during my **AI Web Developer Internship at InAmigos Foundation**. The project focuses on creating an engaging, accessible, and user-friendly platform to showcase the organization's mission, programs, and social impact.

---

## 📖 Overview

This project was built to demonstrate modern frontend web development practices while supporting the digital presence of a non-profit organization.

The website provides visitors with information about the NGO, its initiatives, impact, and opportunities to contribute to social causes.

---

## ✨ Features

* 📱 Responsive Design

* 🖥️ Cross-Browser Compatibility
* ⚡ Fast Loading Structure
